# D&D SRD 5.2 Minimal Rule Pack

First replaceable Rule Pack for the generic Action Resolution and Rule Engines. It supplies identifiers, mappings, abilities, selected skills, DC data, checks, attacks, damage, movement, interaction, inventory, initiative, and a small weapon sample.

It intentionally omits complete spellcasting, full combat rounds, resistances/immunities/vulnerabilities, death saves, and complete conditions. See `LICENSE.md` for the required SRD 5.2 / CC BY 4.0 attribution.
